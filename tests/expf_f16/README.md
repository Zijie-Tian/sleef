# Float16 Exponential Function (expf) 测试

这个测试程序比较自定义实现的NEON float16 expf函数与SLEEF库中的expf函数在float16数据上的计算结果。

## 测试内容

测试程序会比较两种实现在以下几类输入下的计算结果：
- 正常范围内的值：0.0, 0.5, 1.0, 2.0, -0.5 等
- 边界情况：接近溢出/下溢的值
- 特殊值：无穷大、NaN等

## 自定义实现

测试中的自定义实现使用基于NEON指令集的直接float16计算，具有以下特点：
- 使用多项式近似来计算e^r (r是小范围内的值)
- 利用exp(x) = 2^n * exp(r) 的恒等式，其中 x = n*ln(2) + r
- 针对float16的数值特性进行优化
- 处理溢出和下溢情况

## SLEEF实现

SLEEF库没有直接的float16实现，测试中使用的方法是：
1. 将float16转换为float32
2. 使用SLEEF的NEON优化版本 `Sleef_expf4_u10advsimd` 计算
3. 将结果转换回float16

## 环境要求

- ARM处理器（支持NEON指令集）
- 编译器支持armv8.2-a+fp16+simd
- SLEEF库已安装

## 编译和运行

```bash
# 编译测试程序
make

# 运行测试
make run
# 或者直接
./expf_f16_test
```

## 结果解释

测试输出将包含：
- 两种实现的计算结果对比
- 相对误差和ULP差异
- 每个测试点是否通过（相对误差小于阈值）
- 总体通过率和最大误差

如果自定义实现与SLEEF的结果足够接近（相对误差小于0.1%），则视为通过测试。
